# Bilder für Benjamin Gillmann Photography

Bitte fügen Sie Ihre Fotos in die entsprechenden Ordner ein:

## Ordnerstruktur:
- `sport/` - Sportfotografie Bilder
  - sport-1.jpg, sport-2.jpg, sport-3.jpg, etc.
  
- `music/` - Musikfotografie Bilder
  - music-1.jpg, music-2.jpg, music-3.jpg, etc.
  
- `events/` - Eventfotografie Bilder
  - event-1.jpg, event-2.jpg, event-3.jpg, etc.
  
- `portfolio/` - Portfolio Highlight Bilder
  - portfolio-1.jpg, portfolio-2.jpg, etc.

## Empfohlene Bildgrößen:
- Galerie-Bilder: 800x600px (4:3)
- Portfolio-Bilder: 1200x800px
- Optimiert für Web (72 DPI)
- Format: JPG oder PNG
- **Schwarz-Weiß** für beste Darstellung!
